# Bangla-News-Dashboard
A beautiful dashboard showing time and news from Bengali newspapers which can be used as a browser startpage.

![image](https://raw.githubusercontent.com/ffrafat/Bangla-News-Dashboard/main/Demo.jpg)

## Features
   * Full Bengali Language
   * Time
   * Date
   * RSS Feed (3 different sources)
